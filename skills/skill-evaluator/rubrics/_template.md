# Evaluation Rubric: <skill-name>

Use this template to create a rubric for a skill that doesn't have one yet. Replace the example dimensions with ones appropriate to the skill's purpose.

## Scoring Scale

- **5**: Excellent — no meaningful room for improvement
- **4**: Good — minor gaps that don't affect usability
- **3**: Adequate — does the job but noticeably imperfect
- **2**: Below expectations — missing important elements
- **1**: Poor — fundamentally fails this dimension

## Dimensions

Define 5-8 dimensions. Each dimension should be:
- **Observable**: can be judged from the output alone (no speculation about process)
- **Independent**: scoring one dimension shouldn't determine another
- **Relevant**: directly tied to the skill's purpose

| Dimension | Definition | What scores 5 | What scores 1 |
|---|---|---|---|
| Completeness | Covers all key elements of the input | All major points addressed, nothing significant omitted | Misses most key content |
| Accuracy | Factual correctness relative to source material | Every claim traceable to source, no fabrication | Contains unsupported or contradicted claims |
| Structure | Organization and scannability of output | Clear headings, logical flow, easy to navigate | Wall of text, no organization |
| Conciseness | Information density without redundancy | Every sentence adds value, no filler | Repetitive, verbose, padded |
| Actionability | Output is directly usable for its intended purpose | Can be used as-is or with minimal editing | Requires substantial rework to be useful |

## Weighted Scoring (optional)

If some dimensions matter more for this skill, assign weights (must sum to 1.0):

| Dimension | Weight |
|---|---|
| Completeness | 0.25 |
| Accuracy | 0.25 |
| Structure | 0.20 |
| Conciseness | 0.15 |
| Actionability | 0.15 |

**Weighted score** = sum of (dimension score x weight)
